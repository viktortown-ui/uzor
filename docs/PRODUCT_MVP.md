# Product MVP

Одна тема: «Время города». Участник выбирает слой, готовый сигнал, группу, последствие, силу и тип свидетельства. Свободный текст сохраняется только как кандидат и не становится общей картиной автоматически.

Не сделано намеренно: несколько тем, публичный режим, профили, лайки, комментарии, реклама, прогнозы, ИИ-классификация, Canvas/WebGL, тяжелые библиотеки графов.

Язык интерфейса избегает слов «граф», «узел», «ребро», «доказано». Используются «УЗОР», «нить», «ветка», «туман», «развилка», «по откликам».
