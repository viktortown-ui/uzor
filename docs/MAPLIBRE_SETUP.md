# MapLibre setup для карты дельт

Карта Delta MVP работает без токена, регистрации и банковской карты.

- Runtime-библиотека: MapLibre GL JS.
- Базовая карта: OpenFreeMap.
- Стиль по умолчанию: `https://tiles.openfreemap.org/styles/liberty`.
- Опционально можно задать `VITE_MAP_STYLE_URL`, если нужен другой совместимый style URL.
- Attribution нельзя скрывать: он нужен для корректного указания источников картографических данных.

В MVP нет адресного autocomplete и reverse geocoder. Место выбирается кликом по карте или кнопкой «Моё местоположение». Район и текстовый ориентир вводятся отдельно и не заменяют координаты.

В будущем provider или style можно заменить через конфигурацию без изменения бизнес-логики дельт.
